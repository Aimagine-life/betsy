# Betsy — автономный AI-агент с характером

AI-агент с настраиваемой личностью, памятью, инструментами и поддержкой любых каналов общения.

Betsy — это **агент**, а не чат-бот. Она умеет выполнять команды, искать информацию в интернете, работать с файлами, устанавливать себе новые возможности и учиться на каждом разговоре.

## Быстрый старт

```bash
# 1. Установи Node.js → https://nodejs.org

# 2. Запусти Betsy
npx betsy
```

Откроется браузер — следуй шагам настройки (60 секунд).

## Установка на сервер (VPS)

```bash
npx betsy-install
```

Введи IP, логин и пароль — установщик сделает всё сам.

## Что умеет

- **Автономный агент** — выполняет задачи, а не просто отвечает на вопросы
- **Настраиваемая личность** — имя, характер, стиль общения
- **Память и самообучение** — запоминает факты, учится на разговорах
- **Виртуальный браузер** — ищет информацию в интернете (Playwright)
- **Инструменты** — выполняет команды, работает с файлами, HTTP-запросы, SSH
- **Навыки (скиллы)** — создаёт и выполняет повторяющиеся сценарии
- **Мультиканальность** — Telegram, Max, браузер и любые новые каналы
- **Плагины** — голос, видео, селфи
- **Саморасширение** — сама устанавливает новые каналы и плагины

## Архитектура

```
betsy/
├── core/           ← Ядро: agentic loop, LLM, память, скиллы
│   └── tools/      ← Инструменты: shell, files, browser, memory, ssh...
├── channels/       ← Каналы: Telegram, Browser (WebSocket)
├── plugins/        ← Плагины: голос, видео, селфи
└── ui/             ← Визард настройки + дашборд (React)
```

### Agentic Loop

Betsy работает через многоходовой цикл с инструментами:

```
сообщение → LLM → "нужно выполнить команду" → выполняет →
→ LLM → "нужно проверить результат" → проверяет →
→ LLM → "готово" → ответ пользователю
```

### Инструменты (9 штук)

| Инструмент | Что делает |
|---|---|
| `shell` | Выполнение команд в терминале |
| `files` | Чтение / запись / список файлов |
| `http` | HTTP-запросы к API |
| `browser` | Виртуальный браузер — поиск, чтение сайтов, скриншоты |
| `memory` | Поиск и сохранение знаний |
| `npm_install` | Установка npm-пакетов |
| `scheduler` | Планировщик задач |
| `self_config` | Изменение собственных настроек |
| `ssh` | Подключение к серверам |

### Каналы

Все каналы равноправны. Betsy может добавить новый канал сама.

| Канал | Что нужно |
|---|---|
| Браузер | Ничего — работает по умолчанию |
| Telegram | Токен от @BotFather |
| Max | Токен бота |

## Настройка

При первом запуске открывается визард в браузере:

1. **OpenRouter API Key** — ключ для AI (бесплатная регистрация → [openrouter.ai](https://openrouter.ai))
2. **Пароль** — для защиты настроек
3. **Личность** — имя, характер, стиль
4. **Каналы** — какие подключить
5. **Готово** — чат прямо в браузере

## Разработка

```bash

npm run build       # Сборка CLI (tsup)
npm run build:ui    # Сборка UI (vite)
npm run build:all   # Всё вместе
npm test            # Тесты (vitest)
npm run typecheck   # Проверка типов
```
