# Betsy — автономный AI-агент с характером

AI-агент с настраиваемой личностью, памятью, инструментами
и поддержкой любых каналов общения.

## Быстрый старт

1. Скачай Node.js → https://nodejs.org
2. Открой терминал (Windows: Win+R → cmd) и вставь:
   ```
   npx betsy
   ```
3. Откроется браузер — следуй шагам (60 секунд)

## Установка на сервер (VPS)

```
npx betsy-install
```

Введи IP, логин и пароль — установщик сделает всё сам.

## Что умеет

- Автономный агент — выполняет задачи, а не просто отвечает
- Настраиваемая личность и имя
- Память и самообучение
- Виртуальный браузер — ищет информацию в интернете
- Выполняет команды и работает с файлами
- Telegram, Max, браузер — и любые новые каналы
- Плагины: голос, видео, селфи
- Сама устанавливает новые каналы и плагины
- Создаёт и выполняет навыки (скиллы)

## Требования

- Node.js 20+
- OpenRouter API ключ (бесплатная регистрация → [openrouter.ai](https://openrouter.ai))

## Docker

```bash
docker run -d --name betsy -p 3777:3777 -v betsy-data:/root/.betsy aimagine/betsy
```
